# Krishi Connect 🌾

Krishi Connect is a web-based platform that connects farmers (sellers) directly with buyers.  
The project helps farmers sell their products online without middlemen and allows buyers to purchase fresh agricultural products easily.

---

# Features

## Seller Module
- Seller Login & Signup
- Add Products
- Upload Product Images/Videos
- Manage Products
- Edit/Delete Products
- View Orders

## Buyer Module
- Buyer Login
- Search Products
- View Product Details
- Buy Products
- Order Management
- Attractive Dashboard

---

# Technologies Used

- HTML
- CSS
- PHP
- MySQL
- XAMPP Server

---

# Database

The project uses MySQL database with the following tables:

- signup
- products
- product_media
- orders
